## 📋 System Capabilities

### 1️⃣ Interview Modes

#### Mode A: Regular Technical Interview
```
Intake Form → 25 Technical Questions → Behavioral Questions → Scoring → Report
```
- Role-based question selection
- Skill-specific deep dives
- Experience-level scaling
- 16,000+ available questions

#### Mode B: Coding Knowledge Test
```
Intake Form → Select Language & Difficulty → Solve Problem → Auto Evaluation → Report
```
- 350+ real programming problems
- Time-based difficulty (5-15 mins)
- Automated test case validation
- Code quality analysis

#### Mode C: Hybrid Interview (RECOMMENDED)
```
Intake → Technical (30%) → Coding (40%) → Behavioral (30%) → Combined Report
```
- Comprehensive skill assessment
- Balanced evaluation
- Professional-grade assessment
